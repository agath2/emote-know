# emote-know

Educational website illustrating the Facial Action Coding System (FACS) developed by Paul Ekman and Wallace V. Friesen (1978). 

Users can select a basic emotion, explore the Action Units (AUs) that compose its facial expression, and click through individual regions of the face to learn the anatomical basis of each movement.

Spring 2026 PSY22 Emotion: Emotion in Daily Life Project
